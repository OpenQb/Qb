# Qb

What is Qb?
--------------------
Qb is a world for creative people. That world can be filled up with IoT apps or Games or Any kind of Applications.


How can i build app for Qb?
---------------------------
Using QML and Javascript.


Which Qt version is used by Qb?
-------------------------------
